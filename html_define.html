  Easy Learning with HTML "Try it Yourself"

Example

<!DOCTYPE html>
<html>
<head>
  <title>Page Title</title>
</head>
<body>

  <h1>This is a Heading</h1>
  <p>This is a paragraph.</p>

</body>
</html>

Output:

This is a Heading

This is a paragraph.
