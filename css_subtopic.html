CSS Styling

What does CSS stand for?

  • CSS stands for Cascading Style Sheets.
  • It is a stylesheet language used to describe the look and formatting of a document written in HTML.
  • CSS controls layout, colors, fonts, spacing, and moreâ€”separating content from visual presentation.

Example:

<style>
  .styled-box {
    background-color: #e0f7fa;
    color: #006064;
    padding: 20px;
    font-weight: bold;
    border-radius: 10px;
  }
</style>

<div class="styled-box">This box is styled with CSS</div>

Output:

This box is styled with CSS

Explanation:

This CSS example shows how to style a div using a class. The `.styled-box` class gives the div a light blue background, dark teal text, padding, and rounded corners. CSS separates content from design and improves reusability.
