JavaScript

What JavaScript Stands For:

  • JavaScript is a programming language that allows you to create interactive and dynamic content on web pages.
  • Unlike HTML or CSS, JavaScript is not an acronym. It was named to ride on the popularity of Java but is entirely different.

Example:

<script>
  function showMessage() {
    document.getElementById('output').innerText = 'Hello from JavaScript!';
  }
</script>

<button onclick="showMessage()">Click Me</button>
<p id="output"></p>

Output:

Explanation:

This JavaScript example adds interactivity to the page. When the button is clicked, a JavaScript function updates the text inside the paragraph using the DOM.
